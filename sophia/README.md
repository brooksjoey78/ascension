# Deus Ex Sophia - Ascension Phases

This directory contains the deployment phases for Ubuntu Ascension v4.0 - Deus Ex Sophia project.

## Phase Scripts

- **phases-1-3.sh**: Foundation, Truth, and Persistence - PURIFIED
  - Environment sanitization
  - Core installation with enhanced truth module
  - Multi-vector persistence engine

- **phases-4.sh**: Network Intelligence Expansion - PURIFIED & HARDENED
  - Passive intelligence module
  - Active reconnaissance
  - Threat intelligence

- **phases-5.sh**: Advanced Exfiltration Matrix v1.0
  - Quantum encryption layer
  - Multiple exfiltration channels
  - Stealth operations

## Usage

Run as root:
```bash
sudo bash sophia/phases-1-3.sh
sudo bash sophia/phases-4.sh
sudo bash sophia/phases-5.sh
```

## Requirements

- Ubuntu 22.04
- Root access
- Internet connection
